Usage Guidelines

Getting Started with shadcn-svelte

NodeSpace uses professional shadcn-svelte components as the foundation. Import and use these components directly for consistent, accessible UI.

// Import shadcn-svelte components
<script lang="ts">
  import { Button } from "$lib/components/ui/button";
  import { Input } from "$lib/components/ui/input";
  import { Card, CardContent, CardHeader, CardTitle } from "$lib/components/ui/card";
  import { cn } from "$lib/utils.js";
</script>

// Professional UI with your NodeSpace colors
<Card>
  <CardHeader>
    <CardTitle>Settings</CardTitle>
  </CardHeader>
  <CardContent>
    <Input placeholder="Enter text..." />
    <Button variant="default">Save</Button>
  </CardContent>
</Card>

Unified Color System

Use shadcn-svelte color variables for consistency. Your NodeSpace colors are automatically mapped:

/* Use shadcn-svelte color variables */
.my-component {
  background-color: hsl(var(--background));
  color: hsl(var(--foreground));
  border: 1px solid hsl(var(--border));
  border-radius: var(--radius);
}

/* Your NodeSpace colors are automatically mapped */
--primary: 210 100% 40%;      /* Your #007acc blue */
--background: 0 0% 100%;       /* Clean white */
--foreground: 220 13% 10%;     /* Dark text */

Building Custom Components

For NodeSpace-specific features, build custom components using the shadcn-svelte foundation:

// Custom component using shadcn-svelte foundation
<script lang="ts">
  import { Card, CardContent } from "$lib/components/ui/card";
  import { Button } from "$lib/components/ui/button";
  import { cn } from "$lib/utils.js";
  
  export let nodeId: string;
  export let hasChildren: boolean = false;
</script>

<Card class={cn("text-node", hasChildren && "has-children")}>
  <CardContent>
    <!-- Custom NodeSpace logic with professional foundation -->
    <Button variant="ghost">ü§ñ AI Assist</Button>
  </CardContent>
</Card>
