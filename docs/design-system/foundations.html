Body Text (1rem): Regular paragraph text with comfortable line-height of 1.6 for optimal readability. This typography system uses system fonts with monospace fallbacks for precise cursor positioning in text editors.

Small Text (0.875rem): Used for captions, metadata, and secondary information.

Code snippets and technical content 
/* Typography System */
:root {
  /* Font Families */
  --font-sans: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif;
  --font-mono: "SF Mono", Monaco, "Cascadia Code", "Roboto Mono", Consolas, monospace;
  
  /* Font Sizes */

  --text-sm: 0.875rem;  /* 14px */

  --text-xl: 1.25rem;   /* 20px */
  --text-2xl: 1.5rem;   /* 24px */
  --text-4xl: 2rem;     /* 32px */
  
  /* Line Heights */
  --leading-tight: 1.2;
  --leading-normal: 1.5;
  --leading-relaxed: 1.6;
}

Spacing System

Consistent spacing based on an 8px grid system for harmonious layouts and visual rhythm.

üìè Spacing Scale
--spacing-1: 0.25rem (4px)
--spacing-2: 0.5rem (8px)
--spacing-3: 0.75rem (12px)
--spacing-4: 1rem (16px) - Base unit
--spacing-6: 1.5rem (24px) - Node indentation
--spacing-8: 2rem (32px)
üèóÔ∏è Layout Spacing

Node Indentation: 24px (1.5rem) per hierarchy level

Component Padding: 16px (1rem) for comfortable touch targets

Section Spacing: 32px (2rem) between major sections

Line Spacing: 1.6 line-height for optimal readability

Theme System

Automatic theme switching with user preference support. Both light and dark themes maintain accessibility and visual hierarchy.

üé≠ Theme Switching

Theme preference is automatically saved and restored across sessions.

‚ôø Accessibility

WCAG 2.1 AA Compliance: All color combinations meet minimum contrast ratios

Light Theme: 4.5:1 contrast ratio for normal text, 3:1 for large text

Dark Theme: Enhanced contrast with careful color adjustments for reduced eye strain

System Integration: Respects user's OS-level theme preferences
