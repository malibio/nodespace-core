Body Text (1rem): Regular paragraph text with comfortable line-height of 1.6 for optimal readability. This typography system uses Inter font family with clean sans-serif fallbacks for professional knowledge management.

Small Text (0.875rem): Used for captions, metadata, and secondary information.

Code snippets and technical content 
/* Typography System - Inter Font Family */
:root {
  /* Font Families */
  --font-sans: Inter, -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif;
  --font-mono: "SF Mono", Monaco, "Cascadia Code", "Roboto Mono", Consolas, monospace;
  
  /* Font Sizes */
  --text-xs: 0.75rem;   /* 12px */
  --text-sm: 0.875rem;  /* 14px */
  --text-base: 1rem;    /* 16px */
  --text-lg: 1.125rem;  /* 18px */
  --text-xl: 1.25rem;   /* 20px */
  --text-2xl: 1.5rem;   /* 24px */
  --text-4xl: 2rem;     /* 32px */
  
  /* Line Heights */
  --leading-tight: 1.2;
  --leading-normal: 1.5;
  --leading-relaxed: 1.6;
}

Theme System

Automatic theme switching with user preference support. Both light and dark themes maintain accessibility and visual hierarchy.

üé≠ Theme Switching

Theme preference is automatically saved and restored across sessions.

‚ôø Accessibility

WCAG 2.1 AA Compliance: All color combinations meet minimum contrast ratios

Light Theme: 4.5:1 contrast ratio for normal text, 3:1 for large text

Dark Theme: Enhanced contrast with careful color adjustments for reduced eye strain

System Integration: Respects user's OS-level theme preferences
