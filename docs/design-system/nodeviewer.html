Node Type Visual System

Visual differentiation system using color-coded indicators for different content types.

📝 Text Node - Primary content, documentation, and narrative text. Uses green indicators for natural content association.
✅ Task Node - Actionable items, todos, and deliverables. Orange indicators create urgency and attention for action items.
🤖 AI Chat Node - AI-generated content, conversations, and insights. Blue indicators distinguish AI contributions while maintaining trust.
🏷️ Entity Node - Tags, categories, and metadata. Purple indicators provide semantic organization and classification.
🔍 Query Node - Search queries, filters, and dynamic content. Pink indicators highlight interactive and query-based content.

Accessibility & Color Independence

♿ Universal Design
  • Shape + Color: Circular indicators with distinct colors
  • Emoji Support: Users can customize indicators with emojis/icons
  • High Contrast: All colors meet WCAG 4.5:1 contrast ratios
  • Consistent Sizing: 8px circles with subtle glow effects
